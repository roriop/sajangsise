# KAMIS Open API 이용 신청서 초안

신청처: https://www.kamis.or.kr → 고객마당 → Open API → 이용신청 (로그인 필요)
문의: 061-930-9971

KAMIS 신청 폼은 로그인 후 노출되며, 서술형 항목이 폼마다 조금씩 다르다.
아래 문구를 칸에 맞춰 그대로 붙여넣으면 된다. 칸이 없으면 건너뛴다.

---

## 서비스명 / 활용 서비스명
```
사장시세 (sajangsise) — 자영업자 식자재 시세 정보 서비스
```

## 서비스 URL / 활용 사이트
```
https://sajangsise.com
```

## 활용 목적 (한 줄)
```
식당·자영업자가 매일 아침 주요 식자재의 도·소매 시세를 빠르게 확인하도록 KAMIS 가격정보를 가공·시각화하여 제공
```

## 활용 분야 / 활용 내용 (서술형, 길게 받는 칸용)
```
'사장시세'는 식당을 운영하는 자영업자가 매일 아침 30초 안에 오늘의 주요 식자재 시세를
확인할 수 있도록 만든 모바일 우선 웹 서비스입니다.

KAMIS Open API의 일별 품목별 도·소매 가격정보(periodProductList)를 활용하여
채소·축산·수산·곡물/가공·조미료·과일 등 자영업 현장에서 자주 쓰는 약 50개 품목의
가격 추이를 카드와 차트로 제공합니다.

- 매일 1회 KAMIS API를 호출해 당일 가격을 수집하고, 품목별 가격 추이 그래프로 시각화합니다.
- 모든 가격 데이터의 출처가 KAMIS(농산물유통정보)임을 사이트 푸터와 소개 페이지에 명시합니다.
- 비영리·정보제공 목적이며, 회원가입 없이 누구나 무료로 열람할 수 있습니다.

호출은 KAMIS 정책에 맞춰 초당 5회 미만으로 제한하며, 하루 1회 배치 수집 방식으로
서버 부하를 최소화합니다.
```

## 출처 표시 문구 (사이트에 게시 — About/푸터)
```
본 서비스의 가격 정보는 KAMIS(농산물유통정보, www.kamis.or.kr) Open API를 기반으로 합니다.
```

## 신청 후 받을 값 (도착하면 GitHub Secrets에 등록)
- `KAMIS_API_KEY` — 인증 Key (p_cert_key)
- `KAMIS_API_ID` — 요청자 id (p_cert_id)

등록 위치: GitHub repo `roriop/sajangsise` → Settings → Secrets and variables → Actions
(cron 워크플로에서 사용. 로컬 `.env`에도 동일 키로 넣되 절대 커밋 금지.)
